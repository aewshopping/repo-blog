# Drawing an ASCII folder tree diagram

This is an excellent site to draw folder tree diagrams for use in code blocks. And also sharing those diagrams within the url 

https://tree.nathanfriend.com/

```
trunk

    leaf.txt
    leaf.txt
  branch
    flower.txt
```
